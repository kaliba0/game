Le Jeu du Pendu

Un jeu fonctionnel, moderne, joli... et tout cela dans un terminal !

  • Un mot mystère est choisi au hasard.
  • Devinez le mot une lettre à la fois.
  • Chaque erreur ajoute une partie du dessin du pendu.
  • Vous gagnez si vous trouvez le mot avant la fin du dessin.
  • Attention : trop d'erreurs, et c'est perdu !

Comment lancer le jeu ?

  1. Naviguez dans le repertoire du jeu avec :

    cd path/to/your/directory

  2. Installer les dépendances nécéssaires :

    pip -m install requirements.txt

  3. Lancer le jeu :

    python main.py
